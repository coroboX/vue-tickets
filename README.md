# vue_tickets

 - [View on github.io](https://corobox.github.io/vue-tickets/index.html)

## Project setup:
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Technologies:
 - HTML/CSS/JS
 - Sass
 - Vue.js framework
 - based on Vue-cli 3
